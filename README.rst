Python 3 library developed by Shd at SMHI.

- Process high resolution oceanographic sensor-data (eg. CTD (SeaBird, Rinco), MVP (AML).

Installation
------------
CTDpy can be installed from PyPI with pip:

.. code-block:: bash

    pip install ctdpy
